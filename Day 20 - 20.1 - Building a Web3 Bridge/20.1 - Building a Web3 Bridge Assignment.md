## Core Assignment — Build an End-to-End Web3 Bridge

In this assignment, you'll build a complete, working cross-chain bridge system from scratch. This includes smart contracts, a bridge operator service, and a frontend interface. By the end, you'll have a fully functional bridge that transfers tokens between Ethereum Sepolia and Base Sepolia testnets.

Use **Foundry** for smart contracts, **Node.js/TypeScript** for the bridge operator, and **React** for the frontend.

### Common Mistakes to Avoid

- ❌ **Not waiting for confirmations**: Always wait for block finality
- ❌ **Missing replay protection**: Use nonce/transfer ID tracking
- ❌ **External calls before state updates**: Update state first to prevent reentrancy
- ❌ **No error handling**: Wrap all async operations in try-catch
- ❌ **Hardcoded values**: Use environment variables
- ❌ **No logging**: Log all operations for debugging
- ❌ **Ignoring gas prices**: Monitor and adjust gas prices
- ❌ **No monitoring**: Set up alerts for operator issues
