## 1. What We're Building

A **Staking Contract** where people can come and stake native ETH.

### Real-Life Analogy

Think of staking like a **bank savings account**:
- You deposit money (stake ETH)
- Money is locked for a period
- You earn interest (token emissions)
- You can withdraw your principal + interest

### Use Case

Staking ETH over to a new blockchain/protocol in exchange for tokens of the new blockchain.

**Example**: Stake ETH to receive protocol governance tokens over time.

## 4. Building the Buggy Staking Contract

Let's intentionally create a buggy version to demonstrate testing and upgrades.

### StakingContract.sol (Buggy Version)

```solidity
// SPDX-License-Identifier: Unlicense
pragma solidity ^0.8.13;

contract StakingContract { 
    uint256 public totalStaked;
    mapping(address => uint256) public stakedBalances;

    function stake(uint256 amount) public payable {
        require(amount > 0, "Amount must be greater than 0");
        require(msg.value == amount, "Amount must be equal to msg.value");
        totalStaked += amount;
        stakedBalances[msg.sender] += amount;
    }

    function unstake(uint256 amount) public payable {
        require(amount <= stakedBalances[msg.sender], "Not enough balance");
        
        // 🐛 BUG: Only returns half the amount!
        totalStaked -= amount / 2;
        stakedBalances[msg.sender] -= amount / 2;
        payable(msg.sender).transfer(amount / 2);
    }
}
```

### The Bug

```
User stakes: 10 ETH
User unstakes: 10 ETH
User receives: 5 ETH ❌ (should get 10 ETH!)
```

## 7. Making It Upgradable with Proxy

To enable smooth upgradability, we introduce a proxy contract.

### StakingProxy.sol

```solidity
pragma solidity ^0.8.0;

contract StakingProxy {
    uint256 public totalStaked;
    mapping(address => uint256) public stakedBalances;
    
    address public implementation;

    constructor(address _implementation) {
        implementation = _implementation;
    }

    fallback() external payable {
        // Forward ALL calls to the implementation contract
        (bool success, ) = implementation.delegatecall(msg.data);
        require(success, "Delegatecall failed");
    }
    
    receive() external payable {}
}
```

### How It Works

```
User → StakingProxy.stake(10 ETH)
            ↓
       fallback() triggered
            ↓
       delegatecall to StakingContract
            ↓
       StakingContract code runs
            ↓
       StakingProxy storage updated
```

### Critical: Storage Layout Must Match!

```solidity
// Proxy
uint256 public totalStaked;        // Slot 0
mapping(address => uint256) public stakedBalances;  // Slot 1
address public implementation;     // Slot 2

// Implementation (must match!)
uint256 public totalStaked;        // Slot 0 ✅
mapping(address => uint256) public stakedBalances;  // Slot 1 ✅
// No implementation variable needed in implementation
```

## 9. Adding Token Emissions

Now let's add the ability to reward stakers with ERC-20 tokens!

### Setup for Emissions

```bash
npm init -y
npm install @openzeppelin/contracts
```

Create `remappings.txt`:
```
@openzeppelin/contracts/=node_modules/@openzeppelin/contracts/
```

### Custom Reward Token (ERC20.sol)

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.13;

import "@openzeppelin/contracts/token/ERC20/ERC20.sol";

contract KiratCoin is ERC20 {
    address stakingContract;
    
    constructor(address _stakingContract) ERC20("KiratCoin", "KIRA") {
        stakingContract = _stakingContract;
    }

    modifier onlyContract() {
        require(msg.sender == stakingContract, "Only staking contract");
        _;
    }

    function mint(address to, uint256 amount) public onlyContract {
        _mint(to, amount);
    }

    function updateContract(address newContract) public onlyContract {
        stakingContract = newContract;
    }
}
```

### Why This Design?

- **Centralized Minting**: Only staking contract can mint
- **Upgradable**: Can point to new staking contract
- **Secure**: No one else can create tokens

## 10. Staking With Emissions

### StakingWithEmissions.sol

```solidity
// SPDX-License-Identifier: Unlicense
pragma solidity ^0.8.13;

interface IKiratToken {
    function mint(address to, uint256 amount) external;
}

contract StakingWithEmissions {
    uint256 public totalStake;
    
    mapping(address => uint256) public stakedBalances;
    mapping(address => uint256) public lastStakeTimestamp;
    mapping(address => uint256) public lastClaimTimestamp;
    
    IKiratToken public rewardToken;
    
    uint256 public constant EMISSION_RATE = 1 ether; // 1 token per second per ETH
    
    constructor(IKiratToken _rewardToken) {
        rewardToken = _rewardToken;
    }
    
    function stake(uint256 amount) public payable {
        require(amount > 0, "Amount must be > 0");
        require(msg.value == amount, "Value mismatch");
        
        // Claim pending rewards before updating stake
        if (stakedBalances[msg.sender] > 0) {
            _claimEmissions(msg.sender);
        }
        
        stakedBalances[msg.sender] += amount;
        totalStake += amount;
        lastStakeTimestamp[msg.sender] = block.timestamp;
        lastClaimTimestamp[msg.sender] = block.timestamp;
    }
    
    function unstake(uint256 amount) public {
        require(amount <= stakedBalances[msg.sender], "Insufficient balance");
        
        // Claim all pending rewards first
        _claimEmissions(msg.sender);
        
        stakedBalances[msg.sender] -= amount;
        totalStake -= amount;
        
        payable(msg.sender).transfer(amount);
    }
    
    function getRewards() public view returns (uint256) {
        if (stakedBalances[msg.sender] == 0) {
            return 0;
        }
        
        uint256 timeStaked = block.timestamp - lastClaimTimestamp[msg.sender];
        uint256 rewards = (stakedBalances[msg.sender] * timeStaked * EMISSION_RATE) / 1 ether;
        
        return rewards;
    }
    
    function claimEmissions() public {
        _claimEmissions(msg.sender);
    }
    
    function _claimEmissions(address user) internal {
        uint256 rewards = getRewards();
        
        if (rewards > 0) {
            lastClaimTimestamp[user] = block.timestamp;
            rewardToken.mint(user, rewards);
        }
    }
}
```

### Reward Calculation Formula

```
Rewards = (Staked Amount × Time Staked × Emission Rate) / 1 ether

Example:
- Staked: 10 ETH
- Time: 100 seconds
- Rate: 1 token/second/ETH
- Rewards: (10 ETH × 100s × 1) / 1 ETH = 1000 tokens
```

## 11. Testing Token Emissions

```solidity
// SPDX-License-Identifier: Unlicense
pragma solidity ^0.8.13;

import "forge-std/Test.sol";
import "src/StakingWithEmissions.sol";
import "src/ERC20.sol";

contract StakingWithEmissionsTest is Test {
    StakingWithEmissions stakingContract;
    KiratCoin kiratToken;

    function setUp() public {
        kiratToken = new KiratCoin(address(this));
        stakingContract = new StakingWithEmissions(IKiratToken(address(kiratToken)));
        kiratToken.updateContract(address(stakingContract));
    }

    function testStake() public {
        uint value = 10 ether;
        stakingContract.stake{value: value}(value);
        assert(stakingContract.totalStake() == value);
    }

    function testGetRewards() public {
        uint value = 1 ether;
        stakingContract.stake{value: value}(value);
        
        // Move time forward 1 second
        vm.warp(block.timestamp + 1);
        
        uint rewards = stakingContract.getRewards();
        assert(rewards == 1 ether); // 1 ETH staked × 1 second × 1 token/s/ETH
    }

    function testComplexGetRewards() public {
        uint value = 1 ether;
        
        // Stake 1 ETH
        stakingContract.stake{value: value}(value);
        
        // 1 second passes
        vm.warp(block.timestamp + 1);
        
        // Stake another 1 ETH (now 2 ETH total)
        stakingContract.stake{value: value}(value);
        
        // 1 more second passes
        vm.warp(block.timestamp + 1);
        
        uint rewards = stakingContract.getRewards();
        
        // Calculation:
        // Period 1: 1 ETH × 1 second = 1 token
        // Period 2: 2 ETH × 1 second = 2 tokens
        // Total: 3 tokens
        assert(rewards == 3 ether);
    }

    function testRedeemRewards() public {
        uint value = 1 ether;
        stakingContract.stake{value: value}(value);
        
        vm.warp(block.timestamp + 1);
        stakingContract.claimEmissions();
        
        assert(kiratToken.balanceOf(address(this)) == 1 ether);
    }
}
```

## 12. Advanced Testing with vm.warp

Foundry's `vm.warp` allows time manipulation in tests.

### Understanding vm.warp

```solidity
// Current time
uint256 currentTime = block.timestamp;  // e.g., 1000

// Warp forward 100 seconds
vm.warp(block.timestamp + 100);

// Now
uint256 newTime = block.timestamp;  // 1100
```

### Testing Time-Dependent Logic

```solidity
function testTimeBasedRewards() public {
    uint stakeAmount = 5 ether;
    
    // Stake at time 0
    stakingContract.stake{value: stakeAmount}(stakeAmount);
    
    // Fast forward 10 seconds
    vm.warp(block.timestamp + 10);
    
    uint256 rewards = stakingContract.getRewards();
    
    // Expected: 5 ETH × 10 seconds × 1 = 50 tokens
    assertEq(rewards, 50 ether);
}
```

## 13. Complete Upgrade Flow

### Scenario: Upgrade from Buggy to Fixed + Emissions

1. **Deploy Buggy Contract**
```solidity
StakingContract v1 = new StakingContract();
```

2. **Deploy Proxy**
```solidity
StakingProxy proxy = new StakingProxy(address(v1));
```

3. **Users Stake** (through proxy)
```solidity
proxy.stake{value: 10 ether}(10 ether);
```

4. **Deploy Fixed Contract**
```solidity
StakingContract2 v2 = new StakingContract2();
```

5. **Upgrade Proxy** (would need upgrade function)
```solidity
proxy.setImplementation(address(v2));
```

6. **Later: Deploy Emissions Version**
```solidity
StakingWithEmissions v3 = new StakingWithEmissions(rewardToken);
proxy.setImplementation(address(v3));
```
