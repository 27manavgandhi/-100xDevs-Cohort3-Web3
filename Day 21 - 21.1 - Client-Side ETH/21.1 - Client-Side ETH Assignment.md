## Common Mistakes to Avoid

- ❌ **Not checking allowance before deposit**: Always verify sufficient allowance
- ❌ **Hardcoding contract addresses**: Use environment variables
- ❌ **Ignoring transaction errors**: Implement proper error handling
- ❌ **Not waiting for confirmations**: Wait for block finality
- ❌ **Missing ABI functions**: Ensure ABI includes all needed functions
- ❌ **Incorrect decimal handling**: USDT uses 6 decimals, not 18
- ❌ **No loading states**: Always show loading indicators
- ❌ **Exposing private keys**: Never commit private keys to git
