## Task 10 — Deploy First Solana Program

Create and deploy a simple Solana program.

```bash
# Create library
cargo init hello-solana --lib

# Add Solana
cargo add solana-program
```

Update Cargo.toml:
```toml
[lib]
crate-type = ["cdylib", "lib"]
```

Create the program:
```rust
use solana_program::{
    account_info::AccountInfo,
    entrypoint,
    entrypoint::ProgramResult,
    msg,
    pubkey::Pubkey,
};

entrypoint!(process_instruction);

pub fn process_instruction(
    program_id: &Pubkey,
    accounts: &[AccountInfo],
    instruction_data: &[u8],
) -> ProgramResult {
    // TODO: Add your logic
    msg!("Hello from Solana!");
    msg!("Program ID: {:?}", program_id);
    msg!("Number of accounts: {}", accounts.len());
    msg!("Instruction data length: {}", instruction_data.len());
    
    Ok(())
}
```

```bash
# Build
cargo build-sbf

# Deploy (requires Solana CLI)
solana program deploy ./target/deploy/hello_solana.so
```

**Requirements:**
- Create library crate
- Add solana-program dependency
- Implement entrypoint
- Use `msg!()` macro for logging
- Build with cargo build-sbf
- Deploy to devnet
- Get program ID from deployment
