## 11. Implementing Structs

You can attach functions to struct instances (similar to classes in TypeScript).

### Method with `&self`

```rust
struct Rect {
   width: u32,
   height: u32,
}

impl Rect {
    fn area(&self) -> u32 {
         self.width * self.height
    }
}

fn main() {
    let rect = Rect {
        width: 30,
        height: 50,
    };
    print!("The area of the rectangle is {}", rect.area());
}
```

### Associated Function (No `&self`)

```rust
struct Rect {
    width: u32,
    height: u32,
 }
 
 impl Rect {
     fn print_str() {
        println!("Inside the rect struct");
     }
 }
 
 fn main() {
    Rect::print_str();  // Called on the type, not instance
 }
```

## 12. Enums

Enums enumerate possible variants of a type.

```rust
enum Direction {
    North,
    East,
    South,
    West,
}

fn main() {
    let my_direction = Direction::North;
    move_around(my_direction);
}

fn move_around(direction: Direction) {
    // implements logic to move a character around
}
```

### Why Not Just Strings?

```rust
// ❌ Bad: No type safety
fn move_around(direction: String) {
    if direction == "north" {
        println!("Moving North");
    }
}
// Problem: Can pass "noth", "North", "NORTH", etc.

// ✅ Good: Type-safe
enum Direction { North, East, South, West }
// Can only be one of these 4 variants!
```

### Enums with Data

```rust
enum Shape {
    Circle(f64),              // radius
    Square(f64),              // side length
    Rectangle(f64, f64),      // width, height
}

fn main() {
    let circle = Shape::Circle(5.0);
    let square = Shape::Square(4.0);
    let rectangle = Shape::Rectangle(3.0, 6.0);
}
```

## 13. Pattern Matching

Match expressions let you handle each variant differently.

```rust
enum Shape {
    Circle(f64),
    Square(f64),
    Rectangle(f64, f64),
}

fn calculate_area(shape: Shape) -> f64 {
    match shape {
        Shape::Circle(radius) => std::f64::consts::PI * radius * radius,
        Shape::Square(side_length) => side_length * side_length,
        Shape::Rectangle(width, height) => width * height,
    }
}

fn main() {
    let circle = Shape::Circle(5.0);
    let square = Shape::Square(4.0);
    let rectangle = Shape::Rectangle(3.0, 6.0);

    println!("Area of circle: {}", calculate_area(circle));
    println!("Area of square: {}", calculate_area(square));
    println!("Area of rectangle: {}", calculate_area(rectangle));
}
```

## 14. Error Handling with Result

Rust doesn't have try-catch. Instead, it uses the `Result` enum.

### Result Enum

```rust
enum Result<T, E> {
    Ok(T),
    Err(E),
}
```

### Reading a File

```rust
use std::fs;

fn main() {
    let greeting_file_result = fs::read_to_string("hello.txt");

    match greeting_file_result {
        Ok(file_content) => {
            println!("File read successfully: {}", file_content);
        },
        Err(error) => {
            println!("Failed to read file: {:?}", error);
        }
    }
}
```

### Using `unwrap()`

```rust
use std::fs;

fn main() {
    let greeting_file_result = fs::read_to_string("hello.txt");
    print!("{}", greeting_file_result.unwrap());  // Panics if error!
}
```

> ⚠️ **Warning**: `unwrap()` will crash your program if there's an error!

### Custom Error Type

```rust
use std::fs;

pub struct FileReadError {}

fn main() {
    let contents = read_file("hello.txt".to_string());
    match contents {
        Ok(file_content) => {
            println!("File content: {}", file_content);
        },
        Err(error) => {
            println!("Error reading file: {:?}", error);
        }
    }
}

fn read_file(file_path: String) -> Result<String, FileReadError> {
    let greeting_file_result = fs::read_to_string("hello.txt");
    match greeting_file_result {
        Ok(file_content) => Ok(file_content),
        Err(error) => Err(FileReadError {})
    }
}
```

## 15. Option Enum

The Option enum handles nullability safely (Rust has no `null`!).

```rust
pub enum Option<T> {
    None,
    Some(T),
}
```

### Example: Finding Character

```rust
fn find_first_a(s: String) -> Option<i32> {
    for (index, character) in s.chars().enumerate() {
        if character == 'a' {
            return Some(index as i32);
        }
    }
    return None;
}

fn main() {
    let my_string = String::from("raman");
    match find_first_a(my_string) {
        Some(index) => println!("The letter 'a' is found at index: {}", index),
        None => println!("The letter 'a' is not found in the string."),
    }
}
```

## 17. Generics & Trait Bounds

Generics allow writing flexible, type-safe code.

### Generic Function

```rust
fn first_element<T>(v: Vec<T>) -> Option<T> {
    return v.into_iter().nth(0);
}

fn main() {
    let v = vec![1, 2, 3];
    let v2 = vec![String::from("Harkirat"), String::from("Singh")];
    
    println!("{}", first_element(v).unwrap());
    println!("{}", first_element(v2).unwrap());
}
```

### Trait Bounds

```rust
fn does_exist<T: std::cmp::PartialEq>(v: Vec<T>, element: T) -> bool {
    let mut iter = v.iter();
    while let Some(value) = iter.next() {
        if *value == element {
            return true;
        }
    }
    return false;
}
```

The `T: std::cmp::PartialEq` means: "T must implement the PartialEq trait"

### More Examples

```rust
// Requires Ord trait (ordering)
fn biggest_element<T: Ord>(a: T, b: T) -> T {
    if a > b { a } else { b }
}

// Requires Display trait (printing)
use std::fmt::Display;

fn print_variable<T: Display>(a: T) {
    println!("{}", a);
}
```

## 18. Generics on Structs

```rust
struct Rect<T> {
    width: T,
    height: T,
}

impl<T: std::ops::Mul<Output = T> + Copy> Rect<T> {
    pub fn area(&self) -> T {
        return self.height * self.width
    }
}

fn main() {
    let r = Rect {
        width: 10,
        height: 20
    };

    println!("{}", r.area());
}
```

### Why Two `<T>`s?

- **First** `<T>`: Defines the generic type with trait bounds
- **Second** `impl Rect<T>`: Says this implementation is for `Rect<T>`

## 19. Traits

Traits are like interfaces in other languages.

```rust
trait Shape {
    fn area(&self) -> f32;
}

struct Rect {
    width: f32,
    height: f32
}

impl Shape for Rect {
    fn area(&self) -> f32 {
        return self.width * self.height
    }
}

fn get_area(shape: impl Shape) -> f32 {
    return shape.area()
}
```

Alternative syntax:

```rust
fn get_area<T: Shape>(shape: T) -> f32 {
    return shape.area()
}

// Or with where clause
fn get_area_3<T>(shape: T) -> f32
    where T: Shape 
{
    return shape.area()
}
```

## 20. Macros

Macros generate code at compile time.

### Declarative Macros

```rust
macro_rules! say_hello {
    () => {
        println!("Hello, world!");
    };
}

fn main() {
    say_hello!();
}
```

### Creating Functions with Macros

```rust
macro_rules! create_function {
    ($func_name:ident) => {
        fn $func_name() {
            println!("Hello from {}", stringify!($func_name));
        }
    };
}

create_function!(hello);

fn main() {
    hello!();  // Prints "Hello from hello"
}
```

### Common Macros

- `println!()` - Print with newline
- `vec![]` - Create vector
- `format!()` - Format string
- `assert!()` - Runtime assertion
- `panic!()` - Crash program

## 21. Serde (Serialization)

```bash
cargo add serde serde_json
```

Cargo.toml:
```toml
serde = { version = "1.0", features = ["derive"] }
serde_json = "1.0"
```

### Example

```rust
use serde::{Serialize, Deserialize};

#[derive(Serialize, Deserialize, Debug)]
struct Person {
    name: String,
    age: u32,
}

fn main() {
    let person = Person {
        name: String::from("John Doe"),
        age: 30,
    };

    // Serialize to JSON
    let json_str = serde_json::to_string(&person).unwrap();
    println!("Serialized JSON: {}", json_str);

    // Deserialize from JSON
    let deserialized_person: Person = serde_json::from_str(&json_str).unwrap();
    println!("Deserialized Person: {:?}", deserialized_person);
}
```

## 22. Borsh (Binary Serialization)

Borsh is used heavily in Solana for efficient, deterministic serialization.

```bash
cargo add borsh
```

Cargo.toml:
```toml
borsh = { version = "1.5", features = ["derive"] }
```

### Example

```rust
use borsh::{BorshSerialize, BorshDeserialize};

#[derive(BorshSerialize, BorshDeserialize, Debug, PartialEq)]
struct MyStruct {
    id: u64,
    data: String,
    v: Vec<u32>
}

fn main() {
    let original = MyStruct {
        id: 42,
        data: "Hello, Borsh!".into(),
        v: vec![1, 2, 3]
    };

    let mut buffer: Vec<u8> = Vec::new();
    original.serialize(&mut buffer).unwrap();
    
    let deserialized = MyStruct::try_from_slice(&buffer).unwrap();
    
    assert_eq!(original, deserialized);
    println!("Successfully serialized and deserialized: {:?}", deserialized);
}
```
