## Core Assignment — Explore Key Management Solutions

**Prerequisites:**
- Understanding of public/private key cryptography
- Solana development experience

---

## Task 1 — Try User-Friendly Wallets

Experience different key management approaches:

1. **Tiplink** (https://tiplink.io/)
   - Create a tiplink
   - Fund it with devnet SOL
   - Try ejecting to get private key

2. **Privy** (https://www.privy.io/)
   - Build simple app with Privy login
   - Observe how keys are managed
   - Test social recovery

3. **Phantom Social Login**
   - Login with Google
   - Compare to traditional seed phrase

**Document:**
- UX differences
- Security trade-offs
- When to use each

---

## Task 2 — Implement solana-tss

Follow the TSS workflow:

1. Install solana-tss
2. Generate 2 key shares
3. Create aggregated public key
4. Fund the aggregated address
5. Create a transaction with both parties signing
6. Broadcast the transaction

**Requirements:**
- Complete all 9 steps
- Document each step
- Verify transaction on Solscan

1. Why can't we just "reset password" for blockchain wallets?
