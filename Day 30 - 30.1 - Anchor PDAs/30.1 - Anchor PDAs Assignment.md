## Core Assignment — Build Staking Program with PDAs

Build a complete staking system using Program Derived Addresses for deterministic account management.

**Prerequisites:**
- Completed Day 28 & 29
- Understanding of CPIs and Anchor

---

## Task 1 — Implement Staking Contract with PDAs

Create a staking program where users can:
1. Create PDA staking account
2. Stake SOL
3. Earn points over time
4. Unstake SOL
5. Claim points

**Requirements:**
- Use PDAs for user staking accounts
- Calculate points: 1 point per SOL per day
- Store bump in account
- Handle time-based rewards
- Safe arithmetic (checked operations)

---

## Task 2 — Write Tests

Test all functionality:
- PDA derivation correctness
- Staking/unstaking flows
- Points calculation accuracy
- CPI with PDA signing
