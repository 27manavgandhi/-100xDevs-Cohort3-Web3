### Test Self-Healing

```bash
# Create drift
kubectl scale deployment -n demo-app demo-app --replicas=10

# Wait and verify correction
watch kubectl get deployment -n demo-app demo-app

# Expected: Replicas return to Git-defined count
```

```bash

kubectl get deployment -n demo-app -o yaml | grep replicas
```

### Common Mistakes to Avoid

- ❌ Forgetting to push Git changes before expecting ArgoCD to sync
- ❌ Using HTTP instead of HTTPS for Git repository URLs in private repos
- ❌ Not waiting for ArgoCD's polling interval (default 3 min) with auto sync
- ❌ Applying manifests directly with kubectl instead of via Git/ArgoCD
- ❌ Enabling self-heal without understanding it will revert manual changes
- ❌ Not creating the target namespace (without CreateNamespace=true)
- ❌ Using wrong Kubernetes API server URL in destination
- ❌ Not enabling automated sync when expecting automatic deployments
