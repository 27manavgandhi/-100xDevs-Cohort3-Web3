## Assignment Tasks

### Task 1 — Show All Tokens in Web Wallet

Extend the **OptiMask wallet** you built in Day 3 to show all SPL tokens held by each wallet address.

Steps:
1. For each derived wallet address, call `getTokenAccountsByOwner` via RPC
2. Parse the token accounts to get mint address + balance
3. Display each token's mint, balance, and decimals in the wallet UI

```js
const tokenAccounts = await connection.getTokenAccountsByOwner(
  new PublicKey(walletAddress),
  { programId: TOKEN_PROGRAM_ID }
);

tokenAccounts.value.forEach(({ account }) => {
  const accountData = AccountLayout.decode(account.data);
  console.log(`Mint: ${new PublicKey(accountData.mint).toBase58()}`);
  console.log(`Amount: ${accountData.amount}`);
});
```

**Bonus**: Fetch token metadata (name, symbol, image) from the mint address using Metaplex or Token-22 metadata.

### Task 2 — Token Launchpad Website

Build a **Token Launchpad** web app that lets users create their own SPL token with:

**Required Inputs:**
- Token Name (e.g., "100xCoin")
- Token Symbol (e.g., "100X")
- Image URL (for metadata)
- Initial Supply
- Decimals (0–9)

**Optional Inputs:**
- Freeze Authority (checkbox — enable/disable)
- Metadata URI (JSON following Metaplex standard)

**What it should do:**
1. Connect to Phantom/Solana wallet (use `@solana/wallet-adapter-react`)
2. Create a Token Mint using Token-22 program with metadata enabled
3. Initialize metadata (name, symbol, URI)
4. Create an ATA for the connected wallet
5. Mint initial supply to the ATA
6. Show the mint address and link to Solscan

**Tech Stack Suggestion:**
```bash
npm create vite@latest token-launchpad
npm install @solana/web3.js @solana/spl-token
npm install @solana/wallet-adapter-react @solana/wallet-adapter-wallets
npm install vite-plugin-node-polyfills
```

**Token-22 with Metadata flow:**
```bash
# Via CLI (to understand the flow)
spl-token --program-id TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb create-token --enable-metadata
spl-token initialize-metadata <MINT> <NAME> <SYMBOL> <URI>
spl-token create-account <MINT>
spl-token mint <MINT> <AMOUNT>
```
