### Challenge 2 — Invoke Signed with PDA

Create a program that uses a Program Derived Address (PDA) to sign CPIs.

**Requirements:**
- Derive PDA from seeds
- Use PDA as authority for transfers
- Use `invoke_signed()` instead of `invoke()`

**Example:**
```rust
let seeds = &[
    b"vault",
    ctx.accounts.user.key().as_ref(),
    &[bump],

token::transfer(
    CpiContext::new_with_signer(

            from: ctx.accounts.vault,
```

### Common Mistakes to Avoid

- ❌ Forgetting to include System Program in accounts
- ❌ Not providing enough lamports for rent exemption
- ❌ Wrong account order in CPI invoke
- ❌ Not validating account ownership before CPI
- ❌ Using `init` without providing `payer` and `space`
- ❌ Forgetting to sign with required keypairs
- ❌ Not checking account is signer when required
- ❌ Using wrong program ID in CPI call
